## Description
Server that takes care of getting flights information from different sources and merging them.

## Installation

```bash
# Node 18.12.0 and NPM 8.19.2
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
